INTRODUCTION

Avant toute manipulation de votre site via l’interface il faut tout d’abord vous connecter sur votre site en ajoutant /wp-admin à l’url de votre site.

Ensuite rentrez vos coordonées: Nom d’utilisateur/mot de passe.

Vous vous retrouvez donc sur le tableau de bord de personalisation de Wordpress:

CHANGEMENT DES PHOTOS DU SLIDER

Afin de changer les photos du slider présent sur la page d’acceuil du site, il faut tout d’abord vous rendre dans l’onglet Smart Slider disponible dans le menu de gauche

Vous arrivez donc sur la page de modification du slider de l’accueil, vous voyez donc les différents sliders disponibles:

Cliquez alors sur: Main Slider - Free Full-width pour accéder à la page de modicication du slider de la page d'accueil.

Pour changer de photo, il vous suffit de cliquer sur l’une d’elles et de choisir la nouvelle photo que vous voulez ajouter. Vous pouvez choisir parmi celles déjà ajoutées dans votre site ou alors en télécharger des nouvelles sur l’onglet téléverser des fichiers.

Vous avez la possibilité d’ajouter une photo à celles déjà présentes, Il suffit d’appuyer sur ajouter une diapositive en haut de la liste des photos ( voir Capture ci-dessus ) et d’ajouter une image P.S: il n’est pas nécessaire de mettre une légende et une URL aux images, puisque que seules les images en elles mêmes seront affichées.

Depuis cette oage, vous avez également la possibilité de modifier l'ensemble des paramètres du slider.

CATALOGUE DE PHOTO

AJOUT D’UNE NOUVELLE PHOTO

Pour ajouter une nouvelle photo au catalogue, rendez-vous sur l’onglet produits sur le menu de gauche et cliquer sur Ajouter produit.

Vous arriverez donc sur cette page :

Inscrivez-y les informations relatives à la photo que vous voulez ajouter ( titre, description)

Ensuite, plus bas, vous verrez une section Données produit. (ci-dessous)

Dans le menu déroulant, au lieu de produit simple, sélectionnez produit variable, le visuel de la section va alors changer :

Cochez l’option vendre individuellement. Ensuite sélectionnez l’onglet Attributs dans le menu :

Sélectionnez Format dans le menu déroulant et appuyez sur Ajouter.

Ensuite, cliquez sur Tout sélectionner, les différents formats de photo s’ajoute alors dans le champ Valeurs, puis cocher l’option utilisé pour les variations et enfin cliquer sur enregistrer les attributs.

Pour résumer, nous venons de créer différents “sous-produits” pour cette photo, ainsi, selon le format choisi par l’utilisateur, le prix s’adaptera.

Maintenant, sélectionnez l’onglet Variations. Dans celui-ci, vous allez y rentrer les différents tarifs selon l’attribut Format.

Dans le menu déroulant Ajouter une variation, sélectionnez Créer les variations pour tous les attributs. Et cliquez sur Aller.

Comment supprimer un produit de la liste.

1. Connectez vous à votre compte administrateur Wordpress.

2. Une fois connectée, sur le menu de gauche, aller dans la partie “Produits”

3. Vous allez avoir la liste de tout les produits, vous verrez sur la gauche des images, une case peut etre cocher. Cochez la et au dessus de la liste, vous aurez plusieurs options donc celle de mettre dans la Corbeille.

4. Une fois que vous avez m’y votre produit dans la “Corbeille”, il faut aller dans la “Corbeille” pour que vous puissiez supprimer définitivement le produit. Comme l’indique l’image si dessous.

Comment ajouter un évènement

1. Connectez vous à votre compte administrateur Wordpress.

2. Une fois connectée, sur le menu de gauche, aller dans la partie “Evenements” et ensuite dans la rubrique "ajouter"

3. Une fois que vous êtes dans la partie "ajouter un évènement" plusieurs options s'offrent à vous pour que vous puissiez donner une desription et d'autre renseignements sur l'évènement

4. Une fois que vous avez écrit tout les renseignements, n'oublier pas de cliquer sur le bouton "publié". Une fois fait le bouton passe de publié a Mise à jour si vous voulez modifier des informations de l'évènements .
